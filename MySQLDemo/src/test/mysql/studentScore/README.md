###简单的学生类、删、改、查

##文件和运行环境
	有一个学生实体类、一个Dao层的实现类、一个基于JUnit的测试类、一个mySQL数据文件
	三个类一定要在同一文件夹下
	MYSQL要开启预处理
	测试类的方法都亲测可用

##基本思路
		通过修改实习类Student来对学生的各个数据进行操作。
		因为名字sname可能重名，所以我设置学号sno为唯一主键。（因为学号可能有字母，所以学号sno是String类型的）
